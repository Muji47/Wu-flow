CS 115

introduction to Computer Science 1

An introduction to the fundamentals of computer science through the application of elementary programming patterns in the functional style of programming. Function definition and application. Tracing via substitution. Design, testing, and documentation. Recursive data definitions. Lists and trees. Functional and data abstraction. [Note: See Note 2 above. Offered: F,W,S]
